Com o elixir instalado para criar um projeto utilize 
-> mix new <nome do projeto>

como testar ?
-> mix test

como formatar o código ?
-> mix format

como atualizar o elixir?
asdf(trabalha outras opções de linguagens não só elixir) controlador de versões
necessário instalar plugins sendo para o elixir o erlang 
-> asdf plugin add erlang https://github.com/asdf-vm/asdf-erlang.git

-> asdf
